React To-Do List App

A simple and modern To-Do List application built using React. It allows users to create, read, update, and delete tasks. Includes search functionality and stores tasks using localStorage. Routing is implemented using React Router for page navigation.

Features

  • ✔ Add, Edit, Delete tasks
  • ✔ Search tasks in real-time
  • ✔ localStorage persistence (data saved even after refresh)
  • ✔ React Router for multi-page navigation
  • ✔ Clean UI with responsive design
